Meet Kiro

Your AI assistant and IDE built to enhance developer productivity

Kiro is an intelligent development environment that combines the power of AI with traditional IDE capabilities. We don't write code for you, but we enhance your ability to code well by anticipating needs, making smart suggestions, and letting you lead the way.

Unique Features

Smart Context Awareness

Use #File, #Folder, #Problems, #Terminal, #Git Diff, and #Codebase to give Kiro precise context about your project.

Steering System

Customize Kiro's behavior with team standards, project norms, and additional context through steering files in .kiro/steering/

Spec-Driven Development

Build complex features incrementally with structured specs that formalize design and implementation processes.

Agent Hooks

Automate workflows with hooks that trigger on events like file saves, updates, or manual actions.

Model Context Protocol (MCP)

Extend Kiro's capabilities with MCP tools and servers for enhanced functionality and integrations.

Multi-Modal Support

Work with code, images, and various file types seamlessly within the same interface.

Autonomy Modes

Autopilot Mode

Kiro can modify files within your workspace autonomously, making changes based on your requirements and project context.

Supervised Mode

Review and approve changes before they're applied, giving you full control over what gets modified in your codebase.
